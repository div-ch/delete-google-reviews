# Google Review Removal Landing Page

## Project Overview

Dedicated landing page for businesses that need Google reviews removed.
Owner: Div Churiwal, founder of Nexus Multimedia LLC.
Primary CTA: Book a free consultation.

## Target Clients

Law firms, medical practices, restaurants, contractors, and any business hurt by negative Google reviews.

## Pricing

- $700 to $950 per review removed
- 94% success rate
- Pay only for reviews successfully removed

## Design System

### Colors
- Navy: #0F2D52
- Sky Blue: #2E86C1
- Orange CTA: #E8470A
- White body sections

### Hero
Navy (#0F2D52) to sky blue (#2E86C1) gradient background.

### Typography
- Body: Plus Jakarta Sans (Google Fonts)
- Headings: Fraunces (Google Fonts)

## Tech Stack

- Pure HTML, CSS, and vanilla JS only
- Single index.html file
- No build step, no frameworks, no dependencies beyond Google Fonts

## Animation

- Always use the Motion library for animations in all web projects
- For projects with a build step: npm install motion
- For this project (no build step): load Motion via CDN script tag

## Writing Rules

- No em dashes anywhere in copy or code comments
- No emojis anywhere
- Keep copy professional and direct

## SEO Requirements

- LocalBusiness schema for San Diego, CA
- FAQPage schema
- Service schema with pricing
- Full Open Graph and Twitter Card meta tags
- Canonical tag pointing to nexusmultimedia.com

## Platforms We Remove From

Google, Glassdoor, Avvo, Yelp, Reddit

## Page Sections

1. Fixed nav with logo and Get a Free Quote CTA
2. Hero with headline, subheadline, two CTAs, floating dashboard mockup
3. Platforms we remove from
4. How it works: 3 steps (Submit, We Handle It, Review Removed)
5. Why reviews get removed: policy violation reasons with icons
6. Pricing: pay only for results, 94% success rate
7. Testimonials: 3 client quotes
8. FAQ accordion with 5 questions
9. 3-step intake form (platform/quantity, situation/urgency, contact info)
10. Footer with links and contact info
